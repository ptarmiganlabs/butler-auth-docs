### Version 1.0

This first version of Butler Auth includes support for these authentication providers:

* Google
* Microsoft
* Okta
* Auth0
* Keycloak
* Facebook
* LDAP
* Local file
* Single user

Docker containers are available on [Docker Hub](https://dockerlskdjflskjf) for both Intel and ARM architectures.
